## Overview

We install and test **ambient sensor systems** in real apartments. No cameras, no microphones – only motion, door, and temperature sensors.

![Smart home sensor setup diagram](/images/projects/smart_sensors.jpg)

## What the System Does

- Detects unusual patterns (e.g., no movement for many hours)
- Sends discreet alerts to family or care services
- Monitors room temperature, humidity, and window status
- All data stays locally encrypted

## Real-World Test

A 6-month trial with **5 households** in Rhein-Ruhr region showed:
- 40% faster response to falls or illness
- High user acceptance (no "big brother" feeling)
- Reduced worry among relatives
